1. 修改中文语言的官网样式时，只修改/public/css/cn/目录下的css文件
2. 禁止修改/public/css/global/目录下的css文件
3. 字体样式，使用思源黑体
4. 当用户的要求的中文样式效果，只是通过css修改达不到这种效果时，需要考虑是否需要修改html结构，以达到这种效果


  5. 如果更改了html结构，但是必须保留国外语言的html结构，结构如下所示：
  ```
  {{#if (eq locale 'cn')}}
    中文语言的html结构
    {{else}}
    其他的国外语言的html结构
  {{/if}}
  ```
5. 中文下的更改，一定不能影响其他语言的样式和内容
6. 中文的标识是cn，而不是zh-CN
7. 如果中文情况下需要更改js,必须保留其他语言的js文件，不能直接修改/public/js/global/目录下的js文件，而且使用  {{#if (eq locale 'cn')}} jscode {{/if}}
